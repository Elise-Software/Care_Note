# 2~3분 데모 시연 스크립트

## 0:00–0:20 · 서비스 소개

“엘리제소프트웨어 돌봄 체크보드는 고령 부모의 진료 후 내용을 가족이 바로 확인할 일로 정리하는 서비스입니다. 의료 판단을 새로 만들지 않고, 입력한 자료를 정리·공유하는 도구입니다.”

## 0:20–0:50 · 직접 입력

1. `새 기록 > 직접 입력`에서 혈압약·재방문·주의사항이 있는 가상 메모를 보여 준다.
2. 개인정보 유사 패턴 경고가 가능성 안내와 마스킹으로 표시됨을 설명한다.
3. `분석 실행`을 누른다.

## 0:50–1:20 · 사진과 음성

1. `사진` 탭에서 `IMG02_ct_notice.png` 합성 안내문을 선택한다.
2. Gemma 4가 보이는 텍스트만 추출하고 사용자가 추출문을 수정·확정해야 한다고 설명한다.
3. `음성` 탭에서 합성 WAV를 재생하고 전사문을 검토·확정하는 흐름을 보여 준다.
4. AI가 연결되지 않아도 텍스트 규칙 fallback이 계속 동작함을 설명한다.

## 1:20–1:55 · 체크보드와 충돌

1. `체크보드` 탭에서 복약, 재방문, 검사 준비, 주의사항, 다음 방문 질문 5개 열을 보여 준다.
2. 완료 체크와 담당자 입력, 원문 근거를 시연한다.
3. 7월 28일 메모와 7월 29일 사진 fixture를 함께 분석해 `입력 자료 확인 필요` 경고를 보여 준다.
4. 서비스가 임의로 하나의 날짜를 선택하지 않는다고 설명한다.

## 1:55–2:20 · 공유문

1. `공유문` 탭에서 가족에게 보낼 문장을 보여 준다.
2. 완료 항목 포함 여부를 바꾸고 복사·다운로드 버튼을 시연한다.
3. 충돌이 있으면 공유문에도 확인 필요가 표시됨을 설명한다.

## 2:20–2:40 · 검증과 마무리

1. `검증 정보` 탭에서 20개 가상 텍스트 샘플의 실제 구조화율 95.45%를 보여 준다.
2. AI schema·grounding·conflict 계약 검증은 별도 수치로 관리한다고 설명한다.
3. “입력 원문에 없는 진단·처방을 만들지 않고, 가족이 확인할 일을 놓치지 않도록 돕습니다.”로 마무리한다.
